# IT & Tools Access (Test-only)

## Access Requests
- Use the IT ticket category “Access Request”.
- Specify tool name, role, and justification.
- Standard access turnaround: 1–2 business days.

## Common Tools
- SSO/Identity provider
- Project wiki
- Source control
- Messaging and video conferencing
- Expense portal
- Data tools (role-dependent)

## Device Policies
- Company laptop is mandatory for engineering/data roles.
- Do not use personal devices for Confidential/Restricted data.

## Troubleshooting
For urgent access issues on Day 1:
- Create an IT ticket with “Day1 Access Blocker”
- Also notify the onboarding coordinator in your team channel.
